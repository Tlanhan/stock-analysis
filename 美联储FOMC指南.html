美联储是干什么的？

美联储 = 美国的"央行"，它的唯一武器是"利率"i

🎯 美联储的"双重使命"
美联储的一切行动都围绕国会赋予的两大法定目标：

① 最大就业 — 让想工作的人都能找到工作
② 物价稳定 — 把通胀控制在 2% 左右

这两个目标经常互相矛盾：
降息 → 经济火热 → 就业 ↑
但 → 通胀失控

加息 → 经济降温 → 通胀 ↓
但 → 失业率 ↑

这就是为什么沃什现在左右为难：CPI 4.2% 要求他加息控通胀，但经济放缓又让他不敢轻举妄动。

利率就是这个天平上的砝码。
——加息给经济踩刹车，降息给经济踩油门。

联邦储备委员会i
🏛️ 为什么委员任期长达14年？
7 名委员的任期是 14 年（全美最长之一），而且是错开的——每两年才有一个名额到期。

这个设计是故意的：
14年超长任期 + 错开到期
→ 一位总统最多只能任命2-3名委员
→ 无法在任期内"换血"整个委员会
→ 美联储不受政治选举周期绑架

主席任期单独计算（4年），但即使总统换了主席，委员们依然在那里制衡。

这就是美联储"政治独立性"的制度基石——它不需要讨好任何一个政客。

对比：财政政策（减税/刺激）由国会决定，受选举影响极大；货币政策由美联储独立决策。

7名委员，总统任命。主席：凯文·沃什（Kevin Warsh） — 2026年新任。

所有 7 名委员 + 所有 12 名地区主席都会出席每次 FOMC 会议并参与讨论，但只有 12 人有投票权：

+ 4 名地区主席（每年轮换）
═════════

为什么要轮换？因为如果 12 家地区银行全部投票，纽约和沿海的金融中心话语权会过大。轮换制度让堪萨斯城、明尼阿波利斯这样的"内陆"声音也有机会影响政策。

但注意：非投票成员也能参与讨论和影响共识。很多时候真正的辩论来自非投票成员——因为他们的票虽然不算，但他们的观点会被纳入会议纪要，影响未来决策。

🎯 FOMC的三个武器

联邦基金利率 ⭐⭐⭐⭐⭐ i
🔵 为什么这是"最核心的武器"？
联邦基金利率是整个金融体系的"基准定价锚"。所有的利率——你的房贷利率、企业发债利率、信用卡利率——都是在它基础上加价。

美联储调整它，等于同时调整了全社会的借贷成本：
加息25bp
→ 房贷利率 ↑
→ 企业融资成本 ↑
→ 消费减少 → 经济降温

银行间隔夜拆借利率。直接决定按揭、信用卡、企业贷款利率——最核心的武器。

量化宽松/紧缩(QE/QT) ⭐⭐⭐ i
🟣 QE/QT 到底怎么"印钱"和"收钱"？
美联储通过买卖国债来控制市场上的美元供应量：

QE（量化宽松）= 印钱
美联储从银行手里买入国债，付给银行新创造的美元 → 银行有了更多钱 → 放贷更多 → 股市流动性增加

QT（量化紧缩）= 收钱
美联储把国债卖回给银行（或到期不续）→ 从银行抽走美元 → 市场流动性减少 → 股市资金被抽走
2020年COVID: QE $4万亿 → 股市暴涨
2022年: 开始QT → 股市暴跌

买/卖国债来注入或抽走流动性。QE=印钱，QT=收钱。

前瞻指引 ⭐⭐⭐⭐ i
🩷 为什么"说话"比"行动"更重要？
金融市场交易的不是"现在"，而是"对未来的预期"。如果美联储能让市场相信"明年会降息"，那么即使今天还没降，市场今天就会提前涨——因为资金在抢跑。

这就是为什么沃什在发布会上说的每一句话都会被逐字分析：
沃什说 "we are not in a hurry to cut"
→ 市场解读: "短期不会降息"
→ 债券收益率 ↑ → 科技股 ↓

一句话的威力 = 一次加息的威力。

用语言引导市场预期。沃什说什么，有时比做什么更重要。

🟣 加息怎么传导到企业借钱成本？
美联储加息后，银行间隔夜拆借利率立刻上升，所有贷款利率随之水涨船高：

• 浮动利率债券：利率跟随基准自动上调，利息支出直接增加
• 新发行的固定利率债券：投资者要求更高收益率，企业发债更贵
• 银行贷款：定价 = 基准利率 + 利差，基准涨多少，贷款就贵多少
加息前: 企业发债利率4.5%
加息25bp后: 新债利率4.75%
→ 借100亿，每年多付2500万利息

而且存量债务也会受影响——大量企业债是浮动利率或近期到期的，到期续借只能接受更高利率。

企业利润下降 i
🩷 借钱变贵→利润下降，到底怎么传导？
融资成本上升通过三个渠道侵蚀利润：

渠道1：利息支出增加（直接）
利润 = 收入 − 成本 − 利息，利息变多 → 利润直接缩水。

渠道2：需求萎缩（间接）
消费者房贷变贵 → 可支配收入减少 → 消费降级 → 企业收入下降。

渠道3：投资收缩（远期）
新项目回报率跑不赢融资成本 → 企业砍掉扩产计划 → 未来增长放缓。
例: 某半导体公司
收入100亿 − 成本80亿 − 利息5亿 = 利润15亿
加息后利息涨到8亿
→ 利润 = 100−80−8 = 12亿
→ EPS下降 20%

这就是为什么 EPS（每股收益）是整条传导链的终点。

股票的价值 = 未来所有利润的折现值。科技股（如半导体、光模块）大部分利润在5-10年后，利率越高，未来利润"折算到今天"的缩水就越严重。

而消费品公司（如可口可乐）利润稳定且集中在近期，受利率影响小得多。

🔵 为什么银行在加息时反而受益？
银行的利润 = 贷款利率 - 存款利率（这个差叫"净息差"）。

加息时：
• 贷款利率立刻上调（房贷、企业贷跟着涨）
• 但存款利率调整很慢（银行不会主动给你加利息）
• 结果：银行赚的"差价"变大 → 利润增加
加息前: 贷款5% - 存款1% = 赚4%
加息后: 贷款7% - 存款1.5% = 赚5.5%
→ 银行利润 ↑37.5%
加息: 🟢
降息: 🔴

🏠 房地产为什么怕加息？
房地产对利率敏感，但不如科技股极端，原因有两面：

利空渠道：
• 房贷利率跟随联邦基金利率上升 → 月供变贵 → 购房需求下降

利率6%: 30年房贷月供 $5,996
→ 月供涨42% → 很多人买不起房

缓冲因素（所以只有两个红圈）：
• 有实物资产和租金收入，不是纯靠未来预期定价
• 现金流在近期产生，折现影响比科技股小得多

所以加息对地产是"受伤但不致命"。

🔍 声明里要盯哪些"关键词"？
FOMC声明虽然只有几百字，但每一个词的变化都可能暗藏信号。最有效的方法不是通读全文，而是拿上一次声明对比，看改了什么。

重点盯这几个词：
• "disinflation"（通胀回落中）→ 鸽派信号 🟢
• "firmly committed"（坚定承诺）→ 强硬鹰派 🔴
• "data dependent"（依赖数据）→ 中性，在观望 🟡
• "some further progress"（取得了一些进展）→ 鸽派转向 🟢
3月声明: "inflation remains elevated"
6月声明: "inflation is moderating"
→ 一个词的微妙变化
→ 鹰派基调在软化
→ 利好科技股！

这就是为什么每次声明发布后，财经媒体第一时间发的文章标题都是 "FOMC声明逐字对比"——因为差异就藏在细节里。

→ 第15天又跌2% → 割肉离场

正确做法：等市场找到新平衡点（通常要1-2周），等波动率下降、成交量萎缩，再考虑进场。

"不抄底"不是永远不买，而是"不要在暴跌的第一天买"。
✅159558跌到2.75以下 → 买500股抄底
✅515880跌到1.50以下 → 买1000股
⏰设止损-5%，等市场消化

• 高开卖一半，剩半设条件（上周的教训）i
⚖️ 为什么卖一半而不是全卖或全留？
这是"对冲不确定性"的经典策略——你不知道高开后会涨还是跌，所以两边都下注。

卖一半的好处：
• 如果继续涨 → 你还有一半仓位吃利润
• 如果回落 → 你已经锁定了一半的利润
持仓2000股 @ 3.00
高开3% → 3.09
卖1000股 → 锁定利润 90元
剩1000股 → 设条件单：
涨破3.12 → 再卖500股
跌破3.03 → 全部止盈

上周的教训：全仓没卖，结果高开后回落，利润变亏损。"卖一半"就是给自己的贪婪和恐惧之间留个缓冲。

交易不是赌方向，而是管理概率。
• 手握现金就是最大的优势i
💰 为什么现金是"最大的优势"？
在金融里，现金有一个专业术语："实物期权"（Real Option）。它的价值在于——你可以在任何你想的时候出手。

满仓的人只有一个选择：祈祷涨。空仓的人有无限选择：什么时候买、买什么、买多少，全由你定。
满仓的人遇到暴跌：
→ 只能干瞪眼，甚至被迫割肉
→ 没有子弹补仓摊低成本

空仓的人遇到暴跌：
→ 可以挑最便宜的标的买入
→ 别人的恐慌 = 你的机会

巴菲特的名言："别人贪婪我恐惧，别人恐惧我贪婪"——但前提是你得有现金在手里才能"贪婪"。

现金不是"没投资"，而是"在等待最好的投资"。
• 7月中报窗口才是真正的建仓窗口i
📊 为什么"中报窗口"是更好的建仓时机？
中报窗口有一个FOMC没有的优势：确定性的信息。

FOMC给你的是"预期"和"暗示"——含糊的、可变的。中报给你的是硬数据——营收多少、利润多少、增长多少，全都有数字。
FOMC说 "通胀有风险" → 利空？（不确定）
中报: 营收+30%，净利+45% → 利好！（确定）
→ 市场敢于据此重估股价

而且7月中报窗口还有一个关键特征：业绩超预期的股票会获得资金集中流入（机构调仓），形成持续1-2周的上涨趋势——这比FOMC的一日行情可靠得多。

FOMC是"天气预报"，中报是"实地考察"。盖房子之前，你会信哪个？

💡 为什么6月和7月要一起看？
单独看一个月的概率没有意义——你需要看的是"利率路径"（Rate Path），即利率在未来的变化轨迹。

举个例子：
6月: 98.5%不变 ← 几乎确定不动
7月: 60%不变，40%降息 ← 关键变量！
─────
如果7月降息概率突然从40%飙到70%
→ 说明市场预期美联储即将转向
→ 科技股会提前涨（抢跑）
→ 这才是你该建仓的信号

所以做功课的核心不是"6月会不会动"（基本不会），而是"7月之后市场押注什么"——那才是市场真正在交易的信息。

看概率的变化趋势，比看绝对值更重要。
2
打开 Investing.com 美联储利率观测器（中文版），截图保存当前概率i
📸 为什么要截图保存？
因为概率每天都在变。截图是为了建立一个"基准线"（Baseline），这样FOMC之后你才能量化对比。
FOMC前截图: 7月降息概率 = 40%
FOMC后查看: 7月降息概率 = 15%

→ 你能精确感知到"有多鹰"

没有截图，你只能凭感觉说"好像变鹰了"。有了截图，你能用数字说话："降息概率从40%降到15%。"

数据驱动的决策，永远比情绪驱动的决策靠谱。

🏛️ 为什么要看官方纪要？
FOMC会议纪要（Minutes）在会议后3周发布，看似过时，其实藏着"黄金信息"。

纪要记录的是闭门讨论的细节——谁投了反对票、委员们具体在争论什么、哪些数据让他们改变主意。这些信息在发布会和声明里完全看不到。
声明: "委员会将密切关注通胀"（套话）
纪要: "多位委员对服务业通胀
粘性表示担忧，2名委员倾向
在9月会议讨论加息选项"
→ 这才是真正的信号！

声明是给你看的，纪要是给分析师看的。

🤔 思考题

如果6月FOMC点阵图移除了最后一个降息点，但沃什在发布会上说"通胀正朝着正确方向移动"——你认为市场会涨还是跌？为什么？

💡 提示：市场反应取决于"预期差"i

💡 什么是"预期差"？
预期差 = 实际发生的事 - 市场预期会发生的事

市场涨跌不取决于事情本身好坏，而取决于比预期好还是差。
预期: 超级鹰派（加息！）
实际: 只是偏鹰（不加息，移除降息点）
预期差: 实际没那么鹰 → 比预期"好"
→ 市场涨！

这就是为什么"利好出尽是利空，利空出尽是利好"——因为最坏/最好的预期已经被提前price in了。
而非绝对方向。想想市场已经price in了什么——如果所有人预期超级鹰派，但实际只是"偏鹰"，这是利好还是利空？
