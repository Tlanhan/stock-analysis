## 第一课：美联储是干什么的？

### 一句话理解

> **美联储 = 美国的"央行"，它的唯一武器是"利率"——加息给经济踩刹车，降息给经济踩油门。**

### 它的组织结构

```
美联储体系
├── 联邦储备委员会（7人，总统任命）
│   └── 主席：凯文·沃什（Kevin Warsh）← 2026年新任！
├── 12家地区联邦储备银行
│   ├── 纽约（最重要，执行公开市场操作）
│   ├── 旧金山、芝加哥...
│   └── 各有一票
└── FOMC（联邦公开市场委员会）
    ├── 12人投票（7名委员 + 12家储备行轮值5家）
    └── 每年开8次会，决定利率
```

### FOMC的三个武器

| 武器 | 作用 | 你需要关注的 |
|------|------|------------|
| **联邦基金利率** | 银行间隔夜拆借利率 | ⭐⭐⭐⭐⭐ 最核心！直接决定按揭、信用卡、企业贷款利率 |
| **量化宽松(QE)/紧缩(QT)** | 买/卖国债来注入/抽走流动性 | ⭐⭐⭐ 影响股市流动性 |
| **前瞻指引** | 用语言引导市场预期 | ⭐⭐⭐⭐ 鲍威尔/沃什说什么比做什么更重要 |

## 第二课：利率怎么影响股市？

### 传导链条（最重要的逻辑）

```
美联储加息
  → 企业借钱成本上升
  → 企业利润下降
  → 股票估值下降（特别是成长股）
  → 股价下跌

美联储降息
  → 企业借钱成本下降
  → 企业利润上升
  → 股票估值上升（成长股最受益）
  → 股价上涨
```

### 对不同板块的影响

| | 加息（踩刹车）| 降息（踩油门）|
|--|-------------|-------------|
| **科技/成长股** | 🔴🔴🔴 暴跌（估值依赖未来现金流） | 🟢🟢🟢 暴涨 |
| **银行股** | 🟢 利差扩大利好 | 🔴 利差缩小利空 |
| **房地产** | 🔴 按揭利率上升 | 🟢 按揭利率下降 |
| **消费品** | 🟡 影响较小 | 🟡 影响较小 |
| **黄金/加密** | 🔴 持有成本上升 | 🟢🟢 避险+流动性双重利好 |

> **你本周交易的半导体ETF和通信ETF属于"科技/成长股"，对利率极度敏感。这就是为什么FOMC对你如此重要。**

## 第三课：FOMC会议怎么看？

### 会议流程（两天）

```
第一天（6/17 周三）：
└── 闭门讨论，不公开

第二天（6/18 周四）美东时间：
├── 14:00（北京时间周五凌晨2:00）
│   └── 发布三样东西：
│       1. 利率决议（加/降/不变）
│       2. 政策声明（economic statement）
│       3. 经济预测摘要（SEP）←⭐ 含点阵图！
│
└── 14:30（北京时间周五凌晨2:30）
    └── 沃什新闻发布会（30分钟）←⭐⭐⭐ 最重要！
```

### 你需要盯的三个关键点

#### ① 利率决议——最直接

| 结果 | 概率（当前市场定价）| 市场反应 |
|------|-------------------|---------|
| **维持不变** | **98.5%** | 大概率平静 |
| 降息25bp | 1.5% | 暴涨（不太可能）|
| 加息25bp | ~0% | 暴跌（不太可能）|

> **6月几乎肯定不加息不降息。真正的战场不在利率决议本身，而在后面两样东西。**

#### ② 点阵图（Dot Plot）——最关键 ⭐⭐⭐⭐⭐

**什么是点阵图？**
- FOMC的12个委员每个人在图上画一个点，代表他/她对未来利率的预测
- 每个季度更新一次（3月、6月、9月、12月）
- **6月会议会发布新点阵图！**

**怎么看？**

```
利率 %
4.0% ·  ·        ← 如果点上移到这 = 委员们预计要加息！
3.75% ·  ·  ·  · ← 当前利率区间（3.50-3.75%）
3.50% ·  ·  ·  ·
3.25% ·  ·       ← 如果点下移到这 = 委员们预计要降息
      2026 2027 2028
```

**本次会议的焦点：**

> Reuters 6/4日报道标题：**"先消失的是最后一个降息点，然后是前瞻指引——然后是加息？"**

- 上次（3月）点阵图里还剩**最后一个"降息点"**（即还有委员认为2026年会降息）
- 如果6月这个降息点消失了 → 意味着**所有委员都不认为今年会降息**
- 更极端：如果开始出现"加息点" → **超级鹰派，市场可能暴跌**

**对你来说：**
- 点阵图上移 → 🟡 利空科技股，但可能已被price in
- 点阵图不变 → 🟢 利空出尽，反弹
- 点阵图显示加息可能 → 🔴🔴 科技股可能大跌

#### ③ 沃什新闻发布会——最大变量

**沃什是谁？**
- 2026年新任美联储主席，6/17-18是他**FOMC首秀**
- 以鹰派著称（倾向于加息控制通胀）
- 提出过改革美联储沟通框架：可能**取消点阵图、修改前瞻指引**

**你要听的关键词：**

| 鹰派词汇（利空）🔴 | 鸽派词汇（利好）🟢 |
|------------------|------------------|
| "Inflation risks remain" 通胀风险仍在 | "Disinflation continuing" 通胀继续回落 |
| "Need to be vigilant" 需要保持警惕 | "Patient approach" 耐心的方式 |
| "Not ready to cut" 还没准备好降息 | "Data dependent" 依赖数据（中性偏鸽）|
| "Potential for hikes" 可能需要加息 | "Balanced risks" 风险平衡 |
| 删除"easing bias"措辞 | 保留"easing bias"措辞 |

**最可能的情况：**
- 沃什释放鹰派信号（强调通胀风险）
- 但不立即加息（维持利率不变）
- 点阵图可能移除最后一个降息点
- **= 偏鹰但不极端 = 科技股短期承压但不会崩盘**

## 第四课：当前美联储面临的困局

### 沃什的两难

```
困局：
├── CPI 4.2%（三年新高）→ 需要加息控制通胀
├── 但经济增长放缓 → 加息可能引发衰退
├── 中东油价波动 → 通胀可能继续恶化
├── SpaceX/OpenAI等巨型IPO → 需要稳定的市场环境
└── 他自己的改革议程 → 需要建立信誉
```

### 当前利率状态

| 指标 | 数据 |
|------|------|
| 当前联邦基金利率 | **3.50%-3.75%** |
| 已经维持不变的会议数 | **连续3次**（4月为最近一次）|
| 上次加息 | 2025年 |
| 上次降息 | 2025年初 |
| 市场6月维持不变概率 | **98.5%** |
| 市场12月加息概率 | **54.1%** |

### 市场定价了什么？

> **市场已经price in（定价了）：6月不加息 + 年内不降息 + 12月可能加息。**

这意味着：
- 如果沃什只是"偏鹰" → 市场已经准备好了，可能不会大跌
- 如果沃什"超级鹰派"（暗示7月就可能加息）→ 超出预期，市场可能暴跌
- 如果沃什意外偏鸽 → 惊喜，市场暴涨

## 附：关键术语速查表

| 术语 | 含义 | 对股市影响 |
|------|------|-----------|
| **FOMC** | 联邦公开市场委员会，决定利率的会议 | — |
| **联邦基金利率** | 银行间隔夜拆借利率 | ↑ 利空股市 ↓ 利好股市 |
| **鹰派（Hawkish）** | 倾向加息控制通胀 | 🔴 利空 |
| **鸽派（Dovish）** | 倾向降息刺激经济 | 🟢 利好 |
| **点阵图（Dot Plot）** | 委员们对未来利率的预测 | 上移🔴 下移🟢 |
| **前瞻指引** | 美联储对未来的口头暗示 | 鹰派措辞🔴 鸽派措辞🟢 |
| **SEP** | 经济预测摘要（Summary of Economic Projections）| 含GDP、失业率、通胀、利率预测 |
| **QE/QT** | 量化宽松/紧缩（买/卖国债）| QE🟢 QT🔴 |
| **basis point（bp）** | 基点，1bp = 0.01% | 加息25bp = 利率上升0.25% |
| **Price in** | 市场已经预期并反映了某个事件 | 已price in的事件不再引起大波动 |
| **CPI** | 消费者价格指数，衡量通胀 | CPI超预期→加息预期↑→股市↓ |
| **PPI** | 生产者价格指数，CPI的领先指标 | 同CPI |
| **非农就业** | 每月新增就业人数 | 超预期→经济强→加息预期↑→股市↓ |
