## 事件一：周一（6/29）错过的承接博弈点 ⚠️ 核心

### 1. 当时的盘面（事中视角）

```
时间：6/29 周一 下午 13:00 左右
背景：
├─ 深成指当天走强，但午后 13:00 出现一轮跳水
├─ 深指从高点回落 → 跳水到【开盘价附近】
├─ 159558 半导体设备ETF：跟随深指小幅度跳水，回到【开盘价附近】
└─ 588200 科创芯片ETF：同步跟随，回到【开盘价附近】
```

### 2. 为什么这是博弈点（错过的逻辑）

**深指回到开盘价 = 承接强的信号**，这是这次最核心的认知：

| 观察 | 含义 |
|------|------|
| 深指跳水但只回到开盘价就止跌 | 说明下方有承接，没有破位 |
| 半导体双ETF跟随回到开盘价 | 等于回到了"市场当日的成本线" |
| 开盘价是天然的支撑/锚点 | 回到这里不破 = 主力在维护成本价 |

**博弈逻辑链：**
```
深指跳水 → 跌到开盘价附近止跌 → 说明承接强
   → 半导体ETF同步回到开盘价 = 低吸点
   → 此时建仓/加仓，盈亏比好（止损就在开盘价下方）
   → 周二（6/30）高抛，获利空间更大
```

### 3. 为什么没做（偏离原因）

**我偏离了规划，因为**：
1. **没有"盘中加仓/建仓"的规则**——只有开盘前写卡的"开盘价"和"止损位"，没有覆盖"盘中跳水到某位置怎么办"。
2. **持仓混淆**——159558 和 588200 都是半导体敞口，午后一起下杀时，大脑混乱分不清该加哪只。
3. **决策推到尾盘**——本该在13:00做决定，却拖到尾盘，结果尾盘价格已经回升，低吸窗口关闭。

### 4. 如果当时做了（量化复盘）

```
假设 6/29 13:00 在开盘价附近建仓/加仓：
├─ 159558：开盘价约 4.22，13:00回到4.22附近
│   → 周二 6/30 开盘高开 4.563，高抛到 4.515
│   → 比周一追高建仓的浮盈更厚，操作空间更大
│
└─ 588200：开盘价约 4.72，13:00回到4.72附近
    → 周二 6/30 高开，4.711 卖出
    → 同样获得更好的成本优势
```

**结论**：周一13:00的承接点如果用了，周二的高抛会从容得多——因为成本更低，不怕小幅回撤，可以等到更好的卖点，而不是10分钟就因为"盘面差"全清。

## 教训 → 规则升级（最核心）

### 规则名称：深指开盘价承接法则 ⭐ 新增

**触发条件：** 持仓标的或目标标的，盘中跟随深指跳水，回落到**当日开盘价附近**且**未跌破**

**必须确认（三个条件全满足才动手）：**
1. ✅ 深指跳水但止跌于开盘价（说明大盘承接强，未破位）
2. ✅ 标的同步回到开盘价附近（回到市场成本线）
3. ✅ 成交量未放巨量（缩量回踩=健康，放量破位=放弃）

**执行动作：**
```
三条件全满足 → 在开盘价附近建仓/加仓
├─ 止损：设在开盘价下方 1%（破开盘价即认错）
├─ 仓位：金字塔分批（-1.5%买1000、-3%买2000...）
└─ 止盈：次日高抛或等MA5
```

### 规则名称：盘中加仓三铁律 ⭐ 新增

1. **持仓相关性检查**：不同时持有2只相关系数>0.85的ETF（如159558+588200）。要做半导体，选1只。
2. **决策时间窗**：跳水出现后 **15分钟内**必须做决定，不准推到尾盘。
3. **分批进场**：跳水点不一次性梭哈，按金字塔分批（-1.5%/-3%/-5%），每档配对卖出计划。

### 下次遇到同类场景的应对（具体到动作）

1. **看到深指午后跳水**：立刻看标的有没有回到开盘价——回到就是博弈点
2. **回到开盘价附近**：确认承接强（未破位）→ 15分钟内分批建仓，止损设在开盘价下方
3. **持仓只有1只半导体ETF**：避免相关性混乱，决策清晰
4. **次日高抛**：成本优势在手，可以从容等更好的卖点

## 🧠 补充复盘（6/30 盘后）—— 问题的真正根源 ⭐

> 盘后冷静下来，发现档案原分析还差一层。真正的根源不是"承接点没买"，而是**持仓设计本身**。

### 更深的认知：单变量持仓法 / 核心博弈聚焦法

**这次的核心错误**：昨天（6/29）买入的真正赌注是"寒武纪突破"，而寒武纪在 588200 里。但同时持了 159558（半导体设备），两只高度相关的票午后一起下杀，大脑过载、分不清该看哪只，导致混乱中早早全清。

**正确的做法应该是**：
```
竞价阶段：
├─ 159558（半导体设备）→ 按跌停价直接挂卖，竞价清掉
└─ 588200（科创芯片，含寒武纪）→ 只留这一只

开盘后：
├─ 压力大幅降低（只剩一只，波动清晰可读）
├─ 专注观察寒武纪走势（真正的赌注）
│   ├─ 若突破向上 → 持股，等第一次回调低吸
│   └─ 若突破失败 → 再决定
└─ 单变量决策 = 清晰、可执行
```

### 提炼规则：核心博弈聚焦法 ⭐ 新增（比承接法则更深一层）

**触发条件**：当你"赌一个具体逻辑"时（如"寒武纪要突破""存储要涨价"）

**必须执行**：
1. **只持含该逻辑的那只票**，清掉所有相关但非核心的干扰项
2. 持仓 = 赌注，让盘面波动只反映"你的赌注对错"，不要混入第二个变量
3. 单变量持仓 = 决策清晰 = 不被波动牵着走

### 为什么这是"真正的根源"（修正原归因）

- 6/15 无计划踏空 → 没有仓位计划
- 6/30 承接点没买 → 表面是"没规则"，**实际是"两只票一起跳，大脑算不过来"**
- 两次犯错的共同根因：**持仓没有聚焦到"你真正赌的那个变量"上**

> **状态差的时候做不了计算——但如果是单变量持仓，根本不需要计算，看寒武纪一眼就够了。**

### 档案归因修正

| 层次 | 原归因 | 修正后 |
|------|--------|--------|
| 根因1（规则层） | 缺盘中承接加仓规则 | 保留（仍成立） |
| **根因2（结构层，更深）** | — | **持仓设计没聚焦核心赌注，多变量导致大脑过载** |

**行动优先级**：下次赌单一逻辑时，先做"持仓聚焦"（只留含该逻辑的那只），再谈进出场。
