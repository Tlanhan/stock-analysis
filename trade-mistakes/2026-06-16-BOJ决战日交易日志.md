## 五、核心认知升级

### 认知1：利好保质期递减定律

```
6/12 周五：有色利好 → 涨一整天（保质期1天）
6/15 周一：美伊利好 → 涨一整天但缩量（保质期1天但衰减）
6/16 周二：BOJ偏鸽 → 涨6分钟（保质期6分钟）

→ 利好脉冲时间逐日递减
→ 当利好保质期<1小时 = 市场进入"垃圾时间"
→ 垃圾时间不做交易
```

### 认知2：跷跷板效应

```
科技和有色是跷跷板：
  6/15：科技暴涨5-7%，有色跟涨4% → 科技领先
  6/16：科技回调0.6%，有色翻绿-0.7% → 两边都跌

→ 跷跷板不是"一边涨一边跌"，而是"轮流涨轮流跌"
→ 中证500/1000在跷跷板中两边都吃=全天候选择
```

### 认知3：诱多识别

```
14:14科创偷拉的特征：
  ① 只有科创在拉，上证/有色不跟
  ② 在重大事件（14:30内田记者会）前偷拉
  ③ 12分钟快速拉升0.7-1%
  ④ 拉完就跌

→ 这是典型的"尾盘诱多"=主力在给明天出货做准备
→ 识别方法：只拉一个板块+大盘不跟+重大事件前=诱多
```

### 认知4：缩量=大资金不参与

```
510300 沪深300全天仅45亿（昨天50+亿）
→ 大资金/北向/机构不参与
→ 没有大资金参与的市场=游资主导
→ 游资主导的市场=暴涨暴跌+不持续
```
