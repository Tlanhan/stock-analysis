## 4. 交易节奏崩坏的五个症状

### 症状①：建仓正确→持仓失控

| 阶段 | 表现 |
|------|------|
| 7/1建仓 | ✅ 方向对、单标的、低位重仓拉成本 |
| 7/2持仓 | 🔴 满仓扛-7.70%，没止损 |
| **诊断** | **建仓有计划，持仓无纪律。止损单没挂=裸奔。** |

### 症状②：亏损后强行操作

| 阶段 | 表现 |
|------|------|
| 7/2深套后 | 没有立刻认错，而是扛着 |
| 7/3做T | 半仓T+0做对了，但仓位残留 |
| 7/6切仓 | 亏损状态下换标的 |
| **诊断** | **亏损状态下不该做决策。7/2深套后应"挂条件单+关软件"，而不是继续手动操作。** |

### 症状③：切仓=凑数交易变种

```
7/6: 588170卖飞 → 立刻切仓到159558
  ↓
表面看是"换标的"，实际是：
  ① 亏损后急于扳回（报复性交易的前兆）
  ② 违反单标的专注（6月刚修复的原则）
  ③ 切仓后立刻满仓30000股（仓位凭感觉）
  ↓
切仓和6月的"凑数"本质一样：都是"不该有的仓位变动"
```

### 症状④：决策-执行时间差

```
7/6晚 22:00  决策完成（参数4.12/4.17/3.95全部算好）
  ↓
7/6晚 22:30  困了，想着"明早再挂"
  ↓
7/7早 09:00  忘记价位，忙乱
  ↓
7/7早 09:25  竞价恐慌手动割3.990
  ↓
时间差 = 11小时
代价 = -6330元
```

**这是最痛的一点：决策完全正确，参数完全正确，但"困了没挂"这11小时的时间差，吃掉了-6330元。**

### 症状⑤：卖在全天最低区

| 日期 | 卖出价 | 全天最低 | 收盘价 | 卖在什么位置 |
|:---:|:---:|:---:|:---:|---|
| 7/6 | 1.191 | 1.149 | 1.192 | **最低区** |
| 7/7 | 3.990 | 3.990(开盘=最低) | 4.137 | **最低点** |

**连续两天卖在全天最低区。这不是巧合，是"恐慌手动操作"的必然结果——人在恐慌时永远卖在最低点。**

## 8. 错误根因分析

### 三层归因：执行层 + 规则层

#### 执行层（最深）

**条件单使用率=0，是本周-1.8万亏损的直接原因。**

| 场景 | 条件单该做什么 | 实际做了什么 | 代价 |
|------|---------|---------|:---:|
| 7/2暴跌 | 4.049自动止损 | 满仓扛 | -1万 |
| 7/6卖飞 | 回落止盈自动卖 | 手动卖飞 | -2000 |
| 7/7割肉 | 3.95未破持有 | 手动割3.990 | -6330 |

**三次"该挂没挂"，三次手动操作，三次卖在最低区。** 条件单的核心价值不是"参数多准"，而是**"剥离情绪"**——人在恐慌时永远做不出好决策，条件单是"恐慌时替你执行理性的工具"。

#### 规则层

**7/6的"切仓"暴露了单标的专注原则的漏洞。**

6月修复的"单标的专注"针对的是"同时持有多只"。但7/6的切仓是"卖一个立刻换另一个"——**形式上单标的，本质上还是在亏损状态下强行操作**。

规则需要补充：**亏损状态下禁止切仓换标的。** 亏损→空仓冷静→重新研究→再建仓，不能亏损→切仓→继续亏。

#### 策略连贯性断裂（E8）

```
7/1: 计划持有588170，止损4.049
  ↓ 7/2暴跌没执行
7/3: 改成做T（策略变了）
  ↓ 
7/6: 卖飞+切仓到159558（标的变了）
  ↓
7/7: 算好参数又没挂（执行断了）
```

**一周内策略变了3次、标的变了2次、执行断了3次。这就是策略连贯性断裂。**

## 9. 教训 → 规则升级

### 下次遇到同类场景的应对（具体到动作）

#### 规则1：条件单使用率必须≥90%

**这是本周最大的教训。** 条件单不是"可选工具"，是"必选工具"。

- 建仓后立刻挂止损单（不能等）
- 止盈用回落止盈条件单（不能手动卖）
- 条件单使用率 = 本周KPI第一指标

#### 规则2：亏损状态下禁止切仓换标的

```
亏损状态 → 空仓冷却（至少72小时）→ 重新研究 → 再建仓
            ↑
       禁止：亏损 → 切仓 → 继续操作
```

切仓是"凑数交易"的变种。6月修了"同时持有多只"，7月要补上"亏损后禁止换标的"。

#### 规则3：满仓扛暴跌=立即认错信号

7/2满仓扛-7.70%是节奏崩坏的起点。**满仓扛暴跌=止损纪律失效=后续所有操作都会变形。**

```
满仓单日跌>5% → 当天必须挂止损单（不能"再看看"）
  ↓
若已深套 → 挂条件单+关软件（不允许手动操作）
  ↓
72小时内禁止任何新操作（模式G情绪冷却）
```

#### 规则4：决策-执行时间差归零（进阶V，7/7已升级）

```
定参数的那一刻 → 必须同时打开交易软件挂上
不允许"等下再挂/明早再挂/明天再说"
没挂=对话没结束
```

**7/8首次实战验证**：7/7晚22:15定参数1.184→立刻挂单→成功。进阶V第一次跑通。

#### 规则5：复盘时间前置

```
旧：睡前算参数（7/6晚22:00）→ 困了没挂 → 出事
新：晚饭后8-9点完成复盘+定参数 → 立刻挂单 → 睡前不决策
```
