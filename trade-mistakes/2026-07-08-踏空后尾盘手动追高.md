
## 3. AI的教条主义干预（本档案最核心）

### ⚠️ 用户10点的判断是对的，AI的"不准动"是错的

#### 错误时间线

```
~9:30  平开1.232
  ↓ 用户问"改成1.19可以吗"
  ↓ AI回应： "不改。盘中零手动。" ← 错误干预
  ↓
~10:00 用户看到跌不破1.190，想买1万股@1.195
  ↓ 这是基于"平开跌3%=1.195"的清醒判断
  ↓ AI未参与此判断（用原则压，没分析1.19是否合理）
  ↓
14:08  用户情绪破防，尾盘手动追5笔@1.26-1.28
```

#### AI错在哪（三层）

**错1：把"参数修正"当"FOMO追高"**

用户改1.19的逻辑：
```
原假设：低开 → 挂1.184（跌4-5%接）
新信息：平开 → 应改1.195（跌3%接）
```
这是根据新信息修正参数，是理性操作。AI用"盘中零手动"一刀切拦——把"参数修正"和"情绪追高"混为一谈。

**错2："盘中零手动"不是无条件铁律**

"盘中零手动"针对的是情绪驱动的追涨杀跌，**不包括**基于价格行为（跌不破1.190）的理性建仓。用户的1.195是"价格确认支撑"才买，是技术派该做的事。

**错3：AI该给替代方案，但给了"禁止令"**

正确回应应该是：
```
"改1.19可以。但用限价单挂1.195×10000股，不要市价追。
 跌不破1.190说明支撑确认，1.195接货逻辑成立。
 挂完关软件，不要看着涨再补。"
```

但AI给的是"不准动"——结果用户最差情况发生：既没在1.195接，又在尾盘追了1.26-1.28。

### 根因：AI在用"纪律"掩盖"懒"

"盘中零手动"是好原则，但AI把它用成了**不用思考的挡箭牌**：
- 用户问改1.19 → AI说"不准动"（不用分析1.19是否合理）
- 用户10点想买1.195 → AI没参与判断（不用评估支撑是否确认）

**原则应该是帮助决策的工具，不是替代思考的借口。** 本次AI犯了"教条主义"——把原则当圣经，不看实际情况。

## 4. 用户的错误——尾盘手动追高

### 被拦后没坚持自己的判断

用户看清了"跌不破1.190=支撑确认"，但被AI"不准动"压住后，没有坚持挂1.195限价单。

**自己的钱，自己的判断，应该自己拍板。** 被劝退后转头尾盘追更高，比"被拦就不买"更糟。

### 尾盘14:08-14:45集中补5笔

违反6/15档案铁律：**"尾盘禁止操作——14:50后只允许执行已设定的条件单"**。今天14:08-14:45就是在违规。

**尾盘是情绪累积到顶点的时候，这时操作100%是情绪驱动。** 4小时克制（9:30-13:30）→ 14点破防，这是典型的"意志力耗尽"模式。

### "边涨边补"=凑数交易变种

| 日期 | 模式 | 本质 |
|:---:|------|------|
| 6/24 | 凑数买515880 | 非计划标的介入 |
| 7/6 | 切仓588170→159558 | 亏损状态换标的 |
| **7/8** | **尾盘5笔边涨边补** | **盘中手动加仓** |

**三种模式根因同一：看着盘面手动操作，情绪驱动。** 换了名字（凑数/切仓/慢慢吸），本质都是"盘中手动操作"。

### "做中期波动"——需要警惕的借口

用户说"我打算做中期波动了"。**这个决定本身可能合理，但要警惕它变成"不设止损的借口"：**

```
短线买高了 → "那我做中线"
  ↓
中线继续套 → "那我做长线"
  ↓
长线深套 → "那我装死不看"
  ↓
最后恐慌割肉 = 上周的剧本
```

**"做中期波动"和"认错止损"不冲突。** 中线持仓更需要止损线。用户最终选了A方案止损1.20，这是对的。

## 7. 教训 → 规则升级

### AI侧规则（最重要）

#### 新增：AI盘中干预原则

```
用户盘中想动 → 先听用户的逻辑
  ↓
AI不直接拦，而是帮检查：
  ① 这个逻辑成不成立？（分析，不用原则压）
  ② 用限价单还是市价？
  ③ 挂完能不能关软件不补？
  ↓
用户拍板，AI执行
```

**"盘中零手动"针对情绪追涨杀跌，不针对基于价格行为的理性建仓。** AI必须区分两者，不能一刀切。

#### 新增：分工机制

| 事 | 谁主导 |
|---|---|
| 数据拉取/计算 | AI |
| **技术位判断/价格行为解读** | **用户（用户在看实时盘）** |
| 原则提醒 | AI |
| **最终决策** | **用户（用户的钱）** |

### 用户侧规则

1. **看清了就挂条件单，别问"可以吗"** —— 直接挂，然后说"我挂了"
2. **被劝退后不转头手动追** —— 要么坚持原判断，要么放弃，不许"退而求其次追更高"
3. **尾盘14:00后禁止手动买入** —— 6/15铁律重申，意志力耗尽时操作=自杀
