## 三、四大结构性问题诊断（核心）⭐

### 问题1：仓位从来不是"算出来的"，是"凭感觉的"

**反复出现**：
- 6/5：8万重仓（过半仓），没考虑非农
- 6/15：10万资金只投1.5万（不知道投多少）
- 6/16：只剩1000股（卖了不敢加）

**根因**：没有"仓位分级规则"。打多少仓全凭当时情绪——亢奋就满仓（6/5），恐惧就空仓（6/16）。

**应建立的规则**：
```
仓位分级铁律：
├─ 底仓层（主线赌注）：总资金50%，只赌一个明确逻辑
├─ 机动层（做T/加仓）：总资金30%，分批用
├─ 现金层（防御）：总资金20%，不轻易动
└─ 强扰动日（非农/BOJ/FOMC）：底仓不超过50%，留50%现金
```

### 问题2：多变量持仓——"做不了两个票"的根源

**反复出现**：
- 6/15：科创芯片+半导体设备两只分开买，脑子不够
- 6/29-30：159558+588200一起跳，大脑过载

**根因**：同时持有2只高度相关ETF，跳水时大脑算不过来→放弃决策。

**应建立的规则**（已提炼为核心博弈聚焦法，见[#11](https://github.com/Tlanhan/stock-journal/issues/11)）：
```
核心博弈聚焦法：
├─ 赌什么逻辑，就只持含该逻辑的那只票
├─ 竞价直接A杀掉非核心持仓（有浮盈就走，不纠结）
├─ 单变量持仓=看一只票就够了，不需要计算
└─ 例：赌寒武纪→只留588200，清掉159558
```

### 问题3：条件单"想做不敢做"——执行纪律缺失

**反复出现**：
- 6/15：知道该挂条件单（寒武纪3.5跌回弹买1万股），但没做
- 6/12：高开没设回落止盈（[#4](https://github.com/Tlanhan/stock-journal/issues/4)）
- 全月：条件单用得太少

**根因**：对条件单的机制不熟+不信任，导致关键时刻回到"手动盯盘→情绪化"。

**应建立的规则**：
```
条件单三场景铁律：
├─ 场景1-高开止盈：高开3%+ → 卖一半 + 剩余设回落止盈
├─ 场景2-回调建仓：挂"价格触发"单（跌到X且回弹买）
├─ 场景3-止损保护：底仓必设止损单
└─ 铁律：交易前先挂条件单，再谈手动操作
```

### 问题4：缩量调整/红十字星=买点 的认知缺失

**出现**：
- 6/16：半导体设备缩量横盘红十字星，犹豫没买（害怕）

**根因**：只认"突破买"，不认"缩量调整买"，错过健康回调的建仓点。

**应建立的规则**：
```
缩量调整买入法：
├─ 信号：上涨后缩量横盘+红十字星（量缩50%+）
├─ 含义：主力洗盘，健康回调
├─ 执行：撑过15分钟不跌 → 分批介入
├─ 加仓：次日再给位置 → 再加1万股
└─ 止损：破前低或放量破位
```

```
6/9  错过启动   → 割完不知道做啥 ❌
6/11 才关注     → 晚了，买点不好 ❌
6/12 高开止盈   → 没全卖，回吐 ❌
6/15 利好兑现   → 无计划踏空   ❌
6/16 缩量买点   → 害怕没买     ❌
6/29 承接点     → 持仓混乱没买 ❌

每次都是"主线已经动了，你才反应过来"。
原因：没有"主线确认"的监控规则，靠肉眼看盘。
```
