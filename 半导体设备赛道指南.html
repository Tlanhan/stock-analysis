半导体周期：现在在什么位置？ i
🔄 什么是"硅周期"？
半导体行业有一个极强的3-4年周期性——叫"硅周期"。

它的形成机制：
需求暴增 → 芯片涨价
→ 厂商拼命扩产（建晶圆厂）
→ 2年后新产能释放
→ 但需求可能已经回落
→ 供过于求 → 价格暴跌
→ 减产裁员 → 库存出清
→ 然后新一轮需求来了
→ 循环往复

这个周期对设备公司尤其重要——因为设备订单是扩产的直接受益者。周期上行时设备公司最先受益，周期下行时它们也最先被砍单。

你的159558就是"周期放大器"。

风险地图：什么会让你亏钱？

🔄

周期下行 i
🔴 最核心的风险
半导体从来没有"一直涨"。上一次2022年SOX暴跌-35%。

周期下行的传导链：
AI需求放缓
→ 芯片库存堆积
→ 晶圆厂砍产能
→ 停止买设备
→ 设备公司订单暴跌
→ 股价腰斩

对设备公司来说，周期下行比芯片设计公司更惨——因为设备订单是"可选消费"，晶圆厂可以先不买、用旧的。

2022年北方华创从高点回撤超过40%。

这是你最大的风险。

硅周期下行 → 设备订单骤减 → 股价暴跌。2022年SOX跌35%，北方华创回撤超40%。

影响等级: ⭐⭐⭐⭐⭐

产能过剩 i
🏭 2027年会不会产能过剩？
台积电2026年CapEx $520-560亿、三星+128%、全球CapEx+20%达$2,000亿——这么多新工厂同时在建。

新产能集中释放

→ 但需求没跟上

→ 设备订单断崖

类似2007-2008年存储芯片崩盘

历史上的教训：每次"这次不一样"之后，都跟着一次惨烈的产能过剩。

关键观察点：2027年AI芯片需求能否消化这些新增产能。

利好面：

利空面：

总体上短期利好、长期不确定。

这是你无法控制的风险，只能通过仓位管理来应对。

双刃剑：禁运利好国产替代，但技术封锁可能限制发展空间。不可控，只能管理仓位。

估值过高 i
📊 PE 118倍意味着什么？
市盈率(PE) = 股价 / 每股收益。PE越高，说明市场愿意为每$1利润付越多钱。

中证半导体材料设备指数PE一度达到118倍——意味着你花$118买$1的年利润。
正常科技公司PE: 20-40倍
热门赛道PE: 40-80倍
118倍 = 极端泡沫区域
───────────
如果利润增速放缓
→ PE从118回落到40
→ 股价要跌 66%！
（即使利润不变）

这就是"杀估值"——不是公司变差了，而是市场不愿意再付那么高的溢价。

2021年底SOX估值峰值后，2022年就-35%。

指数PE一度达118倍。周期下行时可快速回落至15-25倍——纯杀估值就能腰斩。

影响等级: ⭐⭐⭐⭐
