## 七、条件单设置示意图

### 同花顺/东方财富条件单设置路径

```
交易 → 条件单 → 新建条件单
  ├─ 标的：输入代码（如588000）
  ├─ 条件类型：股价触发
  ├─ 触发条件：≤ 或 ≥ 指定价格
  ├─ 触发后操作：买入 / 卖出
  ├─ 委托数量：填入股数
  ├─ 委托价格：市价 或 限价（建议限价=触发价±0.001）
  └─ 有效期：当日有效
```
